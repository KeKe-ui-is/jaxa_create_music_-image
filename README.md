# JAXA Earth Soundscape (Local AI)
地図クリック → 座標 → 衛星データ（NDVI/LST/降水） → ローカルで
- 画像（Stable Diffusion）
- 音（MusicGen）
を生成する Streamlit アプリです。

---

## 動作環境（重要）
このアプリはローカルAI生成があるため、PC性能差が体験差になります。

### 推奨（快適）
- Windows 11
- NVIDIA GPU（VRAM 8GB以上推奨 / 6GBでも可）
- RAM 16GB以上（推奨 32GB）
- 空き容量 10GB以上（モデルDLで増えます）
- Python 3.10〜3.12
-今回は3.11のバージョンを使用しました
### GPUなし（CPU）でも動く？
- 動きますが、画像/音の生成はかなり遅くなります（仕様です）

■ Set-ExecutionPolicy とは？

PowerShellには「実行ポリシー」という安全機能があります。
これは「スクリプト（.ps1ファイル）を勝手に実行しないようにする」ための仕組みです。

初期設定では、Activate.ps1 のようなスクリプトは
実行がブロックされることがあります。

    Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

このコマンドは：

・Scope Process → 今開いているPowerShellウィンドウだけ有効
・ExecutionPolicy Bypass → 一時的に制限を解除

つまり「このウィンドウの間だけ安全制限を解除する」
という意味です。

PC全体の設定は変更しません。
PowerShellを閉じれば元に戻ります。
